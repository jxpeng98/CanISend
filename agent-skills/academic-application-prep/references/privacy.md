# Privacy Rules

Do not commit real applicant data.

Ignored private paths:

- `profile/`
- `jobs/`
- `job_leads/`
- `.env`

Safe-to-commit paths:

- `prompts/`
- `agent-skills/`
- `templates/`
- `schemas/`
- tests and source code

Never fabricate applicant evidence. Missing evidence should be reported as a gap.
