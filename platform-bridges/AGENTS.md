# Academic Application Prep Agent Bridge

Use the project skill at `agent-skills/academic-application-prep/SKILL.md` before preparing academic job application materials.

Start every workflow by checking the private workspace:

```bash
academic-prep doctor --workspace .
```

If running from outside the workspace, replace `.` with the workspace path.

Core rules:

- Treat `profile/`, `jobs/`, `job_leads/`, `.env`, and generated PDFs as private.
- Do not submit applications, fill portals, scrape full job pages, or answer sensitive declarations.
- Use `academic-prep` CLI commands for workspace actions.
- Read `agent-skills/academic-application-prep/references/platforms.md` for cross-platform guidance.
- Read `agent-skills/academic-application-prep/references/quality-gates.md` before presenting generated materials as ready.
